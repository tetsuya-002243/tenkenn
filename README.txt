橋梁点検 写真野帳アプリ PWA版

【今回の追加】
- ホーム画面追加に対応
- アプリアイコンを追加
- manifest.webmanifest を追加
- service-worker.js を追加
- Safariでホーム画面に追加したとき、アプリ風に起動しやすい設定を追加

【開き方】
1. ZIPを解凍
2. index.html を開く
3. iPadでは、安定利用するには社内サーバーやクラウドに置いたURLで開くのが望ましいです

【iPadでアプリ風にする方法】
1. iPadのSafariでアプリURLを開く
2. 共有ボタンを押す
3. 「ホーム画面に追加」を押す
4. ホーム画面の「写真野帳」アイコンから起動

【注意】
ファイルを直接開いた場合、PWA機能やオフライン機能は制限される場合があります。
ホーム画面追加を安定させるには、HTTPSのURLで公開する必要があります。
